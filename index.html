ID Workspace

Платформа для совместной разработки образовательного контента

IDW позволяет компаниям быстро организовать процесс разработки.

На шаблонах IDW любой сотрудник вашей компании сможет сам создать обучающий курс, не привлекая подрядчиков и дополнительный бюджет.

ID Workspace для разных задач.

  • Руководителям в сфере L&D

    IDW позволит наладить процесс быстрой самостоятельной разработки обучающих курсов и управлять им от планирования до согласования.

    IDW заменит профессиональный конструктор и поможет внутренним экспертам разработать курсы самостоятельно.

  • 01

    API

    Возможности API позволяют интегрировать ID Workspace в сложную корпоративную среду, начиная от систем дистанционного обучения, корпоративных порталов и заканчивая мессенджерами и календарями.

  • 02

    Архитектура

    ID Workspace учитывает требования корпоративных заказчиков: мультидоменная архитектура позволяет создавать аккаунты для отдельных дочерних организаций, имеет возможности по настройке ролей, прав доступа и конфигураций функционала.

  • 03

    Масштабируемость

    За счет современного стека технологий и архитектуры, рассчитанной на большие нагрузки, ID Workspace способен обеспечить быструю работу для большой численности одновременно работающих пользователей в крупной корпоративной структуре.
